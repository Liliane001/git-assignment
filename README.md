# Git Assignment

This assignment demonstrates how to use Git, VS Code, and GitHub to manage a project.

I will use VS Code to create and edit project files. I will use Git commands in the terminal to track changes, stage files, and create commits. Git will help me save checkpoints of my work so I can review changes and manage different versions of the project.

This assignment also demonstrates how GitHub can be used to store a project online, share code, and show the history of commits made during development.
